What is Scratch?

 

A brief explanation to what the Scratch programming lanuage is, and what it can do

Scratch is a drag and drop programming lanuage that was developed by the Lifelong Kindergarten Group at the Massachusetts Insitute of Technology (MIT). It is easy to use and anyone can create games using it within minutes. There are over 8 million users and 10 million shared projects.

You can use Scratch on the online editor or download the editor to your computer. The newest version of Scratch, version 2.0, allows to create almost any type of game (within reason); such as 3D FPS games, Platformers and Racing games.

To start using Scratch you need to either sign up for an account (which is free), or download Scratch. I would recomend downloading Scratch 2.0 rather than an earlier version because Scratch 2.0 allows you to create games that you can not in earlier version. However, I would not use an offline editor at all if you can, and use the online editor instead.
